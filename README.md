# trello_remake

[![Build status](https://ci.appveyor.com/api/projects/status/ope18tkm3t6r64c3?svg=true)](https://ci.appveyor.com/project/Stanislavsus-edu/trello-remake)

[gh-pages](https://stanislavsus-edu.github.io/trello_remake/)

Реализована точная копия [Trello](https://trello.com) с упрощенным функционалом:

1. Нельзя создавать, удалять и перемещать колонки

2. Можно создавать карточки и перемещать их как в рамках одной колонки, так и в другие

3. Можно удалить карточку, нажав на иконку крестик

4. В карточке может храниться только текст
